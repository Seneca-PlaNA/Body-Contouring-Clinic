# Body-Contouring-Clinic

### Run the app on your machine:

#### required installation:
- nodejs
- npm

#### steps:
- `git clone`
- `cd` to `Body-Contouring-Clinic/body-contouring-clinic` directory
- `npm install`
- Front-end (port 3000) : `npm start` 
- Back-end  (port 3001) : `node ./servers/server`
